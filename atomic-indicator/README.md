# 原子指标管理 - 数据资产前端

基于 Vue 3 + TypeScript + Element Plus 的原子指标创建与编辑功能。

## 技术栈

- Vue 3
- TypeScript（严格模式）
- Element Plus
- Vue Router
- Vite
- Vitest

## 功能模块

### 1. 指标元件导航
- 左侧导航：指标 → 指标元件（原子指标、修饰词、时间周期）
- 修饰词、时间周期暂为禁用状态

### 2. 原子指标列表页
- 筛选：指标中英文搜索、状态、业务负责人、技术负责人、资产目录
- 操作：新建原子指标、批量转移负责人、批量添加协作人
- 列表：勾选、指标中文（可跳转详情）、指标英文、状态、关联模型、负责人、创建时间
- 行操作：编辑、更多（转移负责人、管理协作人、下线）
- 权限：编辑/转移负责人仅负责人或指标管理员可操作

### 3. 原子指标新建/编辑页
- 业务信息：指标中文名、英文名、口径描述（含校验）
- 管理信息：资产目录、主体（级联：平台指标/券商指标）、业务/技术负责人
- 技术信息：模型选择、设定表达式（选择模式/输入模式）、适用维度、时间维度

### 4. 原子指标详情页
- 基本信息展示
- 专业信息：技术信息、数据血缘（下游基础指标列表）

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务
npm run dev

# 构建
npm run build

# 运行单元测试
npm test
```

```
    └── atomic-indicator/

        ├── Form.vue      # 新建/编辑页

```
